🟢 2️⃣ Index on createdAt
videoSchema.index({ createdAt: -1 });
This helps when you do:
.sort({ createdAt: -1 })
Without index:
MongoDB must:
Read all documents
Then sort in memory ❌
With index:
MongoDB already keeps data sorted internally ✅
Q: Why use indexes?
Answer:
Indexes improve query performance by allowing MongoDB
 to quickly locate documents without scanning the entire collection.
